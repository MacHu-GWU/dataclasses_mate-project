Install
------------------------------------------------------------------------------

``dataclasses_mate`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install dataclasses_mate

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade dataclasses_mate
